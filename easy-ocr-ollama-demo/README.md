# EasyOCR Demo

[English](#easyocr-demo) | [繁體中文](#easyocr-範例專案)

A simple demonstration project for extracting text from images using EasyOCR, a ready-to-use OCR library with support for 80+ languages.

## Features

- Extract text from images using EasyOCR
- Support for multiple languages (English, Traditional Chinese, and more)
- Easy-to-use Python interface
- Lightweight and efficient text recognition

## Prerequisites

- Python 3.7 or higher
- pip package manager

## Installation

1. **Create a Python virtual environment:**
```bash
python3 -m venv .venv
source .venv/bin/activate  # macOS/Linux
# or
.venv\Scripts\activate     # Windows
```

2. **Install dependencies:**
```bash
pip install -r requirements.txt
```

## Usage

Run the project

```bash
python app.py
```

## Project Structure

```
easy-ocr-ollama-demo/
├── README.md
├── requirements.txt
└── app.py
```

## Supported Languages

EasyOCR supports 80+ languages including:
- English (en)
- Traditional Chinese (ch_tra)
- Simplified Chinese (ch_sim)
- Japanese (ja)
- Korean (ko)
- And many more...
